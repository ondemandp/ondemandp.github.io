# 使用Visual C++ 2010创建第一个C程序

# 新建项目

点击“文件”-“新建”-“项目”。

选择“常规”-“空项目”，并填写项目名称和存储位置，点击“确定”。

# 添加源文件

选中“源文件”，右键点击，选择“添加”-"新建项"。

选择“C++文件”，更改名称并点击“添加”。

# 编辑代码

在右侧的编辑区域中输入代码

#include<stdio.h>
int main()
{
	printf("Hello World\n");
	return 0;
}

按键盘上的Ctrl+F5来运行程序。

我们发现程序窗口一闪而过，这是由于程序已执行完毕而自动关闭。

我们要避免这种情况。添加stdlib.h头文件 并在程序结尾处添加system(pause);语句。

所以程序应该是这样的


 



 



#include<stdio.h>
#include<stdlib.h>
int main()
{
	printf("Hello World\n");
	system("pause");
	return 0;
}
