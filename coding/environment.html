# IDE的选择

目前阶段（包括C二级考试）都是在Windows系统下进行操作的，全国计算机等级考试考试大纲(2018年版）所规定的IDE（集成开发环境）为Microsoft Visual C++ 2010 Express。

# 在Android手机上运行C 程序

点此下载CIDE，使用它你可以在你的手机上编写、调试、运行C程序。

# Visual C++ 2010的安装与配置

# 下载安装

点此下载，双击Visual C++ 2010 Express.exe，一路点击下一步即可完成安装。

# 软件激活

打开Visual C++，点击“帮助”-“注册产品”，

输入6VPJ7-H3CXH-HBTPT-X4T74-3YVY7，点击确定完成软件激活。

# 设置字体和字体大小

点击“工具”-“选项”。

点击“环境”-“字体和颜色”，字体选择“Consolas”，大小选择“16”，点击“确定”。

TIP

Consolas适合用作编程字体的原因： 1.容易区分中英文符号； 2.容易区分i/l，0/o等，不易产生错误； 3.字形好看，给人的压力较小。

# 显示行号

点击“文本编辑器”-“所有语言”，把“行号”勾选上，便于在编译器报错时确定问题出在哪一行。
