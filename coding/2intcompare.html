# 比较两个整数的大小

# 示例代码




 








 





 









#include<stdio.h>
#include<stdlib.h>

int compare(); //声明名为compare的函数

void main()
{
	int a,b,c;
	printf("两个整数比较大小\n请输入第一个数\n");
	scanf("%d",&a);
	printf("请输入第二个数\n");
	scanf("%d",&b);
	c = compare(a,b); //调用名为compare的函数
	printf("较大的那个数是%d\n",c);
	system("pause");
	return;
}

int compare(int num1,int num2) //定义名为compare的函数
{
	int result;
	if (num1 >= num2)
		result = num1;
	else
		result = num2;
	return result;
}

# main函数 和 自定义函数

main函数是程序的入口点，一个工程里有且只能有一个main函数。

为了使代码具有清晰易读、便于维护等优点，

我们常常通过自定义函数的方式来将一部分代码转移出main函数而移入一些其他的自定义函数。

需使用时，在main函数中调用这些函数即可。

# 自定义函数的 定义和声明

定义函数需写明返回类型、函数名称和参数。

声明函数需写明返回类型、函数名称，参数中可不填任何东西。

同一个函数在定义和声明中返回类型和函数名称需相同。

# 返回类型

一个函数可以返回一个值。若return的是整数值就写为int，若不返回值就写为void，以此类推。

# 函数名称

由代码编写者自行决定，但为了提高代码可读性应尽量把名字与它的实际作用联系起来。

# 自定义函数的 调用

参阅示例代码。

调用时把main函数中的a和b的值分别传入compare函数的num1和num2中，

而compare函数的作用是比较num1和num2的大小，

并把较大的那个赋值给compare函数的result中，

接着通过return把result中的值传回main函数，

最终将其值赋值给c。
