# C 中的注释

# 作用

1.记录写此行或此段代码的目的，便于自己在次查阅代码时更快找到当时的思路。

在团队中则便于他人对代码进行修改和维护；

2.使某行或某段代码“失效”，适用于暂时不需要这段代码又不能删去的情景。

特别是在排查代码中问题时有很大的作用。

# 单行注释

// write something here
1

# 多行注释

/*
1
2
3
*/
1
2
3
4
5
